# Curso Completo de Power BI - Jornada de Aprendizado e Projetos

Repositório centralizado para documentar toda a minha evolução, projetos práticos e desafios resolvidos ao longo do curso completo de Power BI. Aqui você encontrará desde as etapas iniciais de extração e tratamento de dados (ETL) até a criação de dashboards dinâmicos e análises avançadas com DAX.
